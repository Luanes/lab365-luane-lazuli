# lab365-luane-lazuli

Repositorio para a construcao de uma pagina hub da disciplina "IA para Desenvolvedores" usando Spec-Driven Development (SDD).

## Como navegar no repositorio

- A especificacao principal fica em [spec.md](spec.md).
- A pagina HTML pronta para o Google Sites esta em [ia-para-desenvolvedores.html](ia-para-desenvolvedores.html).
- Conteudos das aulas apresentadas ficam em [assets](assets) (ex: [assets/aulas/aula1.md](assets/aulas/aula1.md)).
- Materiais de referencia adicionais estao em [design-system](design-system) e [system-design.html](system-design.html).

## Fluxo SDD

- Este repositorio usa SDD: altere primeiro a especificacao e depois sincronize a pagina HTML.
- As informacoes oficiais e de negocio estao documentadas em [spec.md](spec.md).

## Como atualizar a especificacao

- A alteracao pode ser feita manualmente pelo usuario diretamente em [spec.md](spec.md).
- Tambem e possivel utilizar assistentes de IA para editar a especificacao e depois aplicar as mudancas na pagina HTML.
