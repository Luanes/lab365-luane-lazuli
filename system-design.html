IA para DEVs

180h
  • Síncrono
  • Avançado
  • Certificado

Ideal para profissionais da área de tecnologia que buscam evoluir. Domine conceitos, práticas e ferramentas para aprofundar a aplicação de IA no desenvolvimento de soluções em tecnologia.

Confirmar inscrição

Para quem é?

Para quem é?

A trilha IA para DEVs, como o nome sugere, é voltada para profissionais que já atuam na área de Desenvolvimento e querem desenvolver competências técnicas de Inteligência Artificial relacionadas à produtividade na programação.

É uma formação especializada, ideal para quem deseja incorporar um diferencial ao seu currículo, desbravar alternativas de programação com IA e alavancar seu trabalho.

O que você vai aprender

Módulo 1 IA no processo de desenvolvimento de software
72h de duração 100% EaD e síncrona (ao vivo)

Capacitar desenvolvedores a integrar ferramentas de IA generativa em seu fluxo de trabalho diário, desde a escrita de código até testes, documentação, mockups e automações.

Fundamentos de IA:

  • Funcionamento do curso e ferramentas
  • Overview de IA: história, ética e vieses
  • IA Generativa: LLMs, prompts e aplicações práticas

Versionamento e IA para Geração de Código:

  • Engenharia de Contexto
  • Git e GitHub com IA (Copilot, AmazonQ, CodeWhisperer, Plugins VS Code)
  • Engenharia de prompts para Geração de código, Refatoração, Geração de mockups e Automação de CI/CD

IA para Testes:

  • Geração de testes unitários/E2E com IA
  • Testes de API via Postman + IA

IA para Documentação:

  • Documentação automática (Swagger, READMEs, diagramas)
Módulo 2 Agentes de IA para desenvolvimento e IA para qualidade de software
108h de duração 100% EaD e síncrona (ao vivo)
Capacitar desenvolvedores a criarem agentes de IA para automatizar processos de software, integrando plataformas no/low-code e APIs, além de aplicar inteligência artificial para elevar a qualidade, testabilidade e observabilidade de aplicações em produção.

Agentes de IA e Automação:

  • Agentes vs Assistentes
  • Consumo de APIs de IA Generativa
  • Plataformas low-code (N8N, AutoGPT, Flowise)
  • MCP (Model Context Protocol): padronização de integração entre ferramentas e agentes de IA
  • Integração com outras ferramentas (Slack, Discord, etc.) via API e/ou webhooks
  • Casos práticos: Deploy automatizado, ChatOps com Slack/Discord e RAG para documentação

IA no processo de Qualidade de Software:

  • CodeReview
  • IA para Testes de Aceitação
  • Testes Automatizados Inteligentes
  • Detecção de Bugs e Anomalias com IA
  • Previsão de Falhas e Análise Preditiva
  • Geração e Otimização de Casos de Teste
  • Assistentes de QA e DevOps com IA
  • Observabilidade e Qualidade em Produção
  • Plataformas Low-Code / No-Code para QA com IA
