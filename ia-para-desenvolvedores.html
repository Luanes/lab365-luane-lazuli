Aula 1 – IA para Geração de Código: Do Prompt ao Pull Request Profissional

Esta aula foca em como utilizar IA para apoiar todo o ciclo de geração de código, da ideação ao pull request. O conteúdo detalhado será preenchido pelo professor conforme o andamento da disciplina.

Tópicos da Aula 1:

  • A representação do mundo de forma digital
  • O que é o prompt e contexto?
  • Ideação e vibe coding
  • Spec-Driven Development
  • Realizando o seu PR com auxílio de IA

1. A representação do mundo de forma digital

Conteúdo deste tópico será definido pelo professor. Use este espaço para explicar os conceitos principais que conectam o mundo real à sua representação em formato digital.

  • Pontos-chave serão adicionados aqui.

2. O que é o prompt e contexto?

Conteúdo deste tópico será definido pelo professor. Utilize este bloco para detalhar como prompts e contexto influenciam a atuação da IA na geração de código.

  • Pontos-chave serão adicionados aqui.

3. Ideação e vibe coding

Conteúdo deste tópico será definido pelo professor. Aqui podem ser descritas estratégias para ideação, exploração de soluções e "vibe coding" com apoio de IA.

  • Pontos-chave serão adicionados aqui.

4. Spec-Driven Development

Conteúdo deste tópico será definido pelo professor. Use este espaço para explicar como especificações orientam o desenvolvimento auxiliado por IA.

Conteúdo deste tópico será definido pelo professor. Descreva aqui boas práticas para preparar, revisar e submeter pull requests com suporte de ferramentas de IA.
